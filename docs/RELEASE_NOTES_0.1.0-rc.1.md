# 《逆序档案》0.1.0-rc.1

首个完整可玩的发布候选版本。

## 下载游玩

- 推荐下载 `reverse-archive-0.1.0-rc.1-offline.zip`，解压后双击 `Reverse-Archive.html`。
- 也可以只下载 `reverse-archive-0.1.0-rc.1.html`，双击后直接游玩。
- 离线版不需要 Node.js、命令行或网络连接，支持最新版 Chrome、Edge 与 Firefox。
- `SHA256SUMS.txt` 可用于校验下载文件完整性。
- HTML 内嵌第三方许可声明，ZIP 中另附 `THIRD_PARTY_NOTICES.txt`。

## 本次内容

- 18 层三幕战役：9 个普通案件、6 个精英案件、3 个章节 Boss。
- 20 张可独立升级卡牌、11 件遗物、6 个异常事件与三种挑战模式。
- 严格按照过去、现在、未来交错结算，未来行动不能追溯挽救过去失败。
- 商店、休整、删牌、升级、受约束奖励与线索驱动终局。
- 四步首次引导、键盘操作、焦点管理与可关闭合成音效。
- V2 至 V5 存档迁移、损坏回退和非法状态修复。

## 验证

- 69 项自动化测试通过。
- TypeScript 类型检查与生产构建通过。
- 375x812 竖屏和 667x320 横屏基线通过。

## 已知限制

- 存档保存在当前浏览器本地，不会在不同浏览器或设备之间同步。
- 当前音效由 Web Audio 实时合成，尚无环境音乐和配音。
- 本版本为静态 Web 离线版，尚未提供原生桌面安装器。
