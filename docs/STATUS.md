# 当前开发状态

更新时间：2026-07-16

## 发布候选概况

- 产品方向：以“过去 / 现在 / 未来”的严格因果时间轴为唯一战斗核心。
- 战役结构：18 层三幕；9 个普通案件、6 个精英案件、3 个章节 Boss、6 个异常事件。
- 构筑内容：20 张可独立升级的卡牌、11 件遗物、休整、商店、删牌、升级与受约束奖励。
- 叙事闭环：四项可追踪关键线索、两段章节结案、证据数量与钥匙状态驱动的终局说明。
- 体验层：四步首次引导、可重看说明、战斗快捷键、焦点管理、弹窗焦点约束、300% 默认合成音效、声音开关与全局音量控制。
- 恢复能力：V2/V3/V4/V5 存档迁移，坏 V5 自动回退旧存档，非法卡牌/遗物/路线/战斗状态会被修复。

## 规则与路线修正

- 行动与固定事件严格按时代交错结算；未来行动不能追溯性挽救过去失败。
- 每幕包含两场必经普通案件；章节 Boss 会检查本幕此前已结案的战斗，不足 3 场时每少一场真相目标 +4。
- 牌组至少保留 3 张，避免空牌组战斗软锁。
- 烧焦的怀表只在每场案件第一次生效，撤回规划会正确返还其免费机会。
- 事件使时间线或悖论达到失败线时立即结束调查。
- 状态机拒绝陈旧路线选择、伪造奖励、错误商店编辑与重复遗物购买。

## 已知限制

- 批量代理用于回归，不替代真实玩家的理解成本与策略多样性测试。
- 当前音效由 Web Audio 实时合成，没有外部音乐与配音资源。
- 当前发布物是静态 Web 构建，尚未封装原生桌面安装器。
